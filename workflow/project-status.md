# High-level overview of current project status

This project has never been deployed to mainnet yet, but we're deploying to testnet now.

Working towards getting the [MVP](/specs/product/mvp.md) ready for people to look at.

Use the verifier workspace to get info on the current state of what's working and what isn't and so on. A useful shortcut for a high-level report: look at the latest verifier/artifacts/root/*/report.md file (the root apex writes it), or run `npm run verifier:go`.
