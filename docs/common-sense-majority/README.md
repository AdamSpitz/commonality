# Common Sense Majority

Common Sense Majority (CSM) is a movement built on top of the [Commonality](/docs/vision-and-strategy/README.md) platform. It uses Commonality's infrastructure — statements, implication graphs, funding portals, delegation, and content funding — to reveal and organize around hidden cross-partisan common ground.

## Background reading

To understand CSM, you need context on:

  - **The Commonality platform itself:** [docs/vision-and-strategy/](/docs/vision-and-strategy/README.md) — the case for decentralized public-goods funding, and why the mechanisms (assurance contracts, delegation, implication graphs, onchain transparency) work.
  - **How the UI domains relate:** [specs/product/ui-domains.md](/specs/product/ui-domains.md) — CSM is one of four branded surfaces. It depends on Noninflammatory Content (the content mechanism) and Commonality (the funding/conceptspace infrastructure).
  - **Hidden-majority content patterns:** [specs/tech/subsystems/conceptspace/content-patterns/hidden-majority.md](/specs/tech/subsystems/conceptspace/content-patterns/hidden-majority.md) — the thesis that on many polarized issues, a supermajority holds a common-sense position that's invisible because the political system is structured around two loud-extremist-dominated coalitions.
  - **The bridge creator:** [specs/product/bridge-creator.md](/specs/product/bridge-creator.md) — the AI system that synthesizes modified statements and commonality statements from moderate positions on opposing sides.
  - **Organic coalitions:** [docs/vision-and-strategy/why-its-better/organic-coalitions.md](/docs/vision-and-strategy/why-its-better/organic-coalitions.md) — how individualized statements + implication graphs discover common ground that centralized systems destroy.
  - **Key ideas (user-facing explanations):** [docs/key-ideas/](/docs/key-ideas/README.md) — delegation, trust networks, content funding, assurance contracts, etc.
  - **CSM walkthrough:** [docs/use-case-walkthroughs/common-sense-majority.md](/docs/use-case-walkthroughs/common-sense-majority.md) — a concrete scenario of how the common-sense majority becomes visible.
