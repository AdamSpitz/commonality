# Common Sense Majority

Common Sense Majority (CSM) is a movement for revealing and organizing the hidden common ground that already exists across political divides. It uses the Commonality ecosystem without being the same movement as Commonality itself:

- **Civility** funds bridge-building media that lets people hear a strong argument without feeling despised.
- **Tally** lets people sign movement-aligned statements in their own words and see direct plus indirect support.
- **Alignment** provides ongoing cause funding and funding portals.
- **Delegation** provides donor-delegate relationships and delegate track records.
- **Pubstarter** provides one-off project funding for organizing projects.
- **Conceptspace** provides the underlying statement, implication, attester, and trust primitives.
