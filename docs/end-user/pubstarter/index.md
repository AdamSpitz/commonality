# Pubstarter

Pubstarter is for individual public-goods projects: creators describe a project, supporters pledge, and funds move only if the project reaches its threshold.

Start with [fund something](/docs/shared/roles/fund-something.md), [get your project funded](/docs/shared/roles/get-your-project-funded.md), and [assurance contracts](/docs/shared/key-ideas/assurance-contracts.md).
