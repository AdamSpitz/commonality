# Civility

Civility is the end-user documentation home for the Noninflammatory Content UI domain. It focuses on finding, funding, and amplifying content that helps people understand each other without escalating conflict.

Start with the [noninflammatory content walkthrough](/docs/shared/use-case-walkthroughs/noninflammatory-content.md), [content funding](/docs/shared/key-ideas/content-funding.md), and the [CSM overview](/docs/csm/README.md).
