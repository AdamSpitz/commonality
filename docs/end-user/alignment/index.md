# Alignment

Alignment is the cause-funding UI for ongoing public-good funding. Use it to explore causes, review funding portals, and pledge through trusted alignment judgments.

For the underlying ideas, see [funding portals](/docs/shared/key-ideas/assurance-contracts.md), [delegation](/docs/shared/key-ideas/delegation.md), and the [pledge-to-a-cause guide](/docs/shared/roles/pledge-to-a-cause.md).
