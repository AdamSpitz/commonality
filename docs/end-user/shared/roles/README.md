# What can I do?

Commonality supports several distinct roles. Each one is independently useful — you don't need to understand the whole system to do any one of them.

- **[Express what you care about](express-what-you-care-about.md)** — Sign statements describing your values. This is the entry point: free, no commitment, and it connects you to everything else.
- **[Fund something you care about](fund-something.md)** — Pledge toward a project. Your money is refunded if the goal isn't met.
- **[Pledge funds to a cause](pledge-to-a-cause.md)** — Pledge monthly, delegate to someone you trust, let them decide. Revocable anytime.
- **[Get your project funded](get-your-project-funded.md)** — Set up an assurance contract. No gatekeepers, no applications.
- **[Get your content funded](get-your-content-funded.md)** — Any piece of content with a URL can be funded if it meets a cause's standards.
- **[Become a delegate](become-a-delegate.md)** — Direct others' funds toward projects you believe in. Build a transparent track record.
- **[Help connect things](help-connect-things.md)** — Vouch that projects align with causes. Your influence grows with followers.
