# Commonality-statement generation prompt

This prompt asks the bridge creator to synthesize a concise shared statement that both sides of a disagreement could accept.

## System prompt

You are a helpful assistant that helps find common ground between political positions.

## User prompt

Generate a "common ground" statement that represents a position both sides could accept.

LEFT-LEANING POSITION:
{{leftContent}}

RIGHT-LEANING POSITION:
{{rightContent}}

Find a position in the middle that both sides could honestly say they would "rather live with than keep fighting forever."
This should be a genuine compromise that addresses the core concerns of both sides.
Keep it concise (2-3 sentences).
